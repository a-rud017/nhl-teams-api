# API for NHL Teams
This API provides information for National Hockey League teams. Information includes Team Name, Location, Team Division, and Arena Name.

## Link to Project
You can find the API [here](https://nhl-teams-api.cyclic.app/).

## Tech Stack
HTML, NodeJS, Express
